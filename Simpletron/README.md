# Simpletron
Il Simpletrone è un implementazione software del microprocessore.

Le sue componenti sono:

1.Memoria RAM (100 words)
2.Accumulatore
3.Instruction Counter
4.Instruction Register

Il modello è preso dal libro "Il linguaggio C" J.Deitel, M.Deitel (Pearson)

Sono forniti due programmi:
Max: restituisce il massimo tra due numeri
Somma: resituisce la somma di due numeri
